### Sobre mim

Construo produtos digitais do zero. Gosto de entender como as coisas funcionam por baixo dos panos — do kernel ao produto final.

- Desenvolvimento **full-stack** com TypeScript, React e Python
- **Infra própria**: Docker, PostgreSQL, Redis, Cloudflare
- Linux no dia a dia — configs em [dotfiles](https://github.com/gram-ms/dotfiles)
- Quando não estou codando, estou quebrando algum servidor de propósito
